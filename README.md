# README

Rails 7, ReactJs, Devise with JWT auth based application. 

Ruby - 3.0.1
Rails - 7.03

database: Sqlite3

To run the application:

Clone the repository:

```cd rails7_react_devise_esbuild```

```bundle install```

```bundle exec rake db:migrate```

```yarn install```

```bin/dev```

explore app on: http://localhost:3000
